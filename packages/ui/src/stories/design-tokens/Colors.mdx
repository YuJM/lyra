# Colors

7개의 컬러 팔레트, 각 11개의 shade (50-950). OKLCH 색공간으로 지각적으로 균일한 그라데이션을 제공합니다.

## 사용 방법

```css
@import "@comneed/lyra-design-tokens/css";

.my-component {
  background: var(--color-blue-500);
  color: var(--color-gray-50);
  border: 1px solid var(--color-gray-300);
}
```

## OKLCH 색공간이란?

OKLCH는 지각적으로 균일한 색공간으로, 다음과 같은 장점이 있습니다:

- **L (Lightness)**: 밝기 (0-1)
- **C (Chroma)**: 채도 (0-0.4)
- **H (Hue)**: 색상 (0-360도)

동일한 간격으로 변화할 때 시각적으로도 동일한 차이로 인식됩니다.

<div className="code-block">{`/* OKLCH 형식 */
color: oklch(0.623 0.214 249.816);
/*      ↑     ↑     ↑
 *      L     C     H
 */

/* 밝기만 조정 */
oklch(0.623 0.214 249.816) → oklch(0.723 0.214 249.816)

/* 채도만 조정 */
oklch(0.623 0.214 249.816) → oklch(0.623 0.314 249.816)

/* 색상만 회전 */
oklch(0.623 0.214 249.816) → oklch(0.623 0.214 349.816)`}</div>
