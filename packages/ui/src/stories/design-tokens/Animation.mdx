# Animation

애니메이션 관련 디자인 토큰입니다. 일관된 모션과 전환 효과를 제공합니다.

## 사용 방법

```css
@import "@lyra/design-tokens/css";

.my-button {
  transition: all var(--duration-base) var(--easing-easeInOut);
}

.my-button:hover {
  transform: scale(1.05);
}
```

## 애니메이션 원칙

### 1. 속도 선택 가이드

**Instant (75ms)**
- 즉각적인 피드백이 필요한 경우
- 마우스 따라가기, 버튼 눌림

**Fast (150ms)**
- 빠른 반응이 중요한 경우
- 호버 효과, 포커스 전환

**Base (250ms)**
- 일반적인 전환 효과
- 탭 전환, 패널 슬라이드

**Moderate (350ms)**
- 복잡한 전환이나 여러 속성 변경
- 페이드 + 이동, 크기 + 투명도

**Slow (500ms)**
- 강조가 필요한 중요한 변화
- 페이지 전환, 모달 등장

### 2. 이징 선택 가이드

**Linear**
- 로딩 스피너, 진행 바
- 일정한 속도가 필요한 경우

**Ease**
- 범용적인 전환
- 기본 호버 효과

**Ease-In**
- 사라지는 요소
- 화면을 벗어나는 애니메이션

**Ease-Out**
- 나타나는 요소
- 화면에 들어오는 애니메이션
- 가장 자주 사용 (권장)

**Ease-In-Out**
- 순환하는 애니메이션
- 시작과 끝이 명확한 전환

### 3. 접근성 고려사항

<div className="code-block">{`/* 애니메이션 축소 선호 시 */

  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }

## Quick Reference

### Duration

| Token | ms | 사용 예시 |
|-------|----|----|
| `--duration-instant` | 75ms | 즉각 피드백, 버튼 눌림 |
| `--duration-fast` | 150ms | 호버 효과, 포커스 |
| `--duration-base` | 250ms | 일반 전환, 탭 전환 |
| `--duration-moderate` | 350ms | 복잡한 전환, 모달 |
| `--duration-slow` | 500ms | 페이지 전환, 강조 |

### Easing

| Token | Value | 사용 예시 |
|-------|-------|----------|
| `--easing-linear` | linear | 로딩, 진행 바 |
| `--easing-ease` | ease | 범용 전환 |
| `--easing-easeIn` | ease-in | 사라지는 요소 |
| `--easing-easeOut` | ease-out | 나타나는 요소 (권장) |
| `--easing-easeInOut` | ease-in-out | 순환 애니메이션 |
