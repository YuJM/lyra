# Shadows

그림자 관련 디자인 토큰입니다. 깊이와 계층을 표현하는 일관된 그림자 시스템을 제공합니다.

## 사용 방법

```css
@import "@comneed/lyra-design-tokens/css";

.my-card {
  box-shadow: var(--shadow-md);
  border-radius: var(--border-radius-md);
}
```

## 깊이 계층 시스템

그림자는 UI 요소의 시각적 계층을 나타냅니다:

### Level 1 - Small (sm)
- **깊이**: 1-2px
- **용도**: 미묘한 구분, 카드, 버튼
- **예시**: 목록 아이템, 작은 버튼

### Level 2 - Medium (md)
- **깊이**: 4-6px
- **용도**: 부유하는 요소, 드롭다운
- **예시**: 드롭다운 메뉴, 팝오버, 날짜 선택기

### Level 3 - Large (lg)
- **깊이**: 10-15px
- **용도**: 최상위 요소, 모달
- **예시**: 모달 다이얼로그, 전체 화면 오버레이

### Special - Inner
- **깊이**: 내부로 2-4px
- **용도**: 눌린 상태, 입력 필드
- **예시**: 활성화된 버튼, 텍스트 입력

## Quick Reference

| Token | 깊이 | 사용 예시 |
|-------|------|----------|
| `--shadow-sm` | 1-2px | 카드, 버튼, 목록 아이템 |
| `--shadow-md` | 4-6px | 드롭다운, 팝오버, 날짜 선택기 |
| `--shadow-lg` | 10-15px | 모달, 다이얼로그, 사이드 패널 |
| `--shadow-inner` | 내부 2-4px | 눌린 버튼, 입력 필드, 토글 |

## 접근성 고려사항

- **대비 의존 금지**: 그림자만으로 중요 정보를 전달하지 마세요
- **색상 모드**: 다크 모드에서는 그림자를 조정하거나 테두리 사용
- **성능**: 과도한 그림자는 렌더링 성능에 영향
- **일관성**: 같은 계층의 요소는 같은 그림자 사용
