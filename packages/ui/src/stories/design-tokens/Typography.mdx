# Typography

폰트 관련 디자인 토큰입니다. 일관된 타이포그래피 시스템을 제공합니다.

## 사용 방법

```css
@import "@comneed/lyra-design-tokens/css";

.my-text {
  font-family: var(--font-family-sans);
  font-size: var(--font-size-base);
  font-weight: var(--font-weight-medium);
  line-height: var(--font-lineHeight-normal);
}
```

---

## Font Families

### Sans-serif (기본)

시스템 폰트 스택으로 빠른 로딩과 네이티브 느낌을 제공합니다.

<div className="typography-demo">
  <div style={{ fontFamily: 'var(--font-family-sans)' }}>
    <div className="token-name">--font-family-sans</div>
    <div className="token-value" style={{ marginTop: '0.25rem', marginBottom: '1rem' }}>
      system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', ...
    </div>
    <div style={{ fontSize: '2rem', fontWeight: 'var(--font-weight-normal)' }}>
      The quick brown fox
    </div>
    <div style={{ fontSize: '1.5rem', fontWeight: 'var(--font-weight-medium)', marginTop: '0.5rem' }}>
      Jumps over the lazy dog
    </div>
    <div style={{ fontSize: '1rem', fontWeight: 'var(--font-weight-normal)', marginTop: '0.5rem' }}>
      0123456789 !@#$%^&*()
    </div>
  </div>
</div>

### Monospace (코드)

코드, 숫자 정렬 등에 사용합니다.

<div className="typography-demo">
  <div style={{ fontFamily: 'var(--font-family-mono)' }}>
    <div className="token-name">--font-family-mono</div>
    <div className="token-value" style={{ marginTop: '0.25rem', marginBottom: '1rem' }}>
      'Menlo', 'Monaco', 'Courier New', monospace
    </div>
    <div style={{ fontSize: '1.25rem' }}>
      const message = "Hello World";
    </div>
    <div style={{ fontSize: '1rem', marginTop: '0.5rem' }}>
      function calculate(a, b) &#123;<br />
      &nbsp;&nbsp;return a + b;<br />
      &#125;
    </div>
  </div>
</div>

## 사용 예시

### 제목 스타일

<div className="code-block">{`h1 {
  font-family: var(--font-family-sans);
  font-size: var(--font-size-2xl);
  font-weight: var(--font-weight-semibold);
  line-height: var(--font-lineHeight-tight);
}

h2 {
  font-size: var(--font-size-xl);
  font-weight: var(--font-weight-semibold);
  line-height: var(--font-lineHeight-tight);
}

h3 {
  font-size: var(--font-size-lg);
  font-weight: var(--font-weight-medium);
  line-height: var(--font-lineHeight-tight);
}`}</div>

### 본문 스타일

<div className="code-block">{`body {
  font-family: var(--font-family-sans);
  font-size: var(--font-size-base);
  font-weight: var(--font-weight-normal);
  line-height: var(--font-lineHeight-normal);
}

p {
  font-size: var(--font-size-base);
  line-height: var(--font-lineHeight-normal);
}

small, .caption {
  font-size: var(--font-size-sm);
}`}</div>

### 코드 블록

<div className="code-block">{`code, pre {
  font-family: var(--font-family-mono);
  font-size: var(--font-size-sm);
}

.code-block {
  font-family: var(--font-family-mono);
  font-size: var(--font-size-base);
  line-height: var(--font-lineHeight-normal);
}`}</div>

## Quick Reference

### Font Size Scale

| Token | px | rem | 사용 예시 |
|-------|----|----|----------|
| `--font-size-sm` | 14px | 0.875rem | 캡션, 메타 정보 |
| `--font-size-base` | 16px | 1rem | 본문 |
| `--font-size-lg` | 18px | 1.125rem | 강조 텍스트 |
| `--font-size-xl` | 20px | 1.25rem | 소제목 |
| `--font-size-2xl` | 24px | 1.5rem | 제목 |

### Font Weight

| Token | Value | 사용 예시 |
|-------|-------|----------|
| `--font-weight-normal` | 400 | 본문 |
| `--font-weight-medium` | 500 | 강조, 레이블 |
| `--font-weight-semibold` | 600 | 제목, 헤딩 |

### Line Height

| Token | Value | 사용 예시 |
|-------|-------|----------|
| `--font-lineHeight-tight` | 1.25 | 제목, 헤딩 |
| `--font-lineHeight-normal` | 1.5 | 본문, 단락 |
