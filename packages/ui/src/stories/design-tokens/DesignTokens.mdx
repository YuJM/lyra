# Design Tokens

Lyra 디자인 시스템의 Primitive 디자인 토큰입니다. W3C DTCG 표준을 준수하며, 일관된 디자인 언어를 제공합니다.

## 📦 설치 및 사용

```bash
pnpm add @lyra/design-tokens
```

### CSS에서 사용

```css
@import "@lyra/design-tokens/css";

.my-component {
  padding: var(--size-4);
  background: var(--color-blue-500);
  border-radius: var(--border-radius-md);
  box-shadow: var(--shadow-md);
}
```

### JavaScript/TypeScript에서 사용

```typescript
import tokens from '@lyra/design-tokens/js';

const primaryColor = tokens.ColorBlue500;
const baseSize = tokens.Size8;
```
