# Size (2px Unit System)

모든 size 토큰은 2px 단위로 증가하여 일관된 spacing rhythm을 제공합니다.

## 사용 방법

```css
@import "@lyra/design-tokens/css";

.my-component {
  padding: var(--size-4);
  margin: var(--size-8);
  gap: var(--size-2);
  width: var(--size-32);
}
```

## 2px 단위의 이점

### 1. 일관된 Rhythm
모든 요소가 2px 그리드에 정렬되어 시각적 일관성이 향상됩니다.

### 2. 유연성
충분히 세밀한 간격 조절이 가능하면서도 관리하기 쉽습니다.

### 3. 반응형 대응
rem 단위로 사용자 브라우저 설정에 반응합니다.

### 4. 수학적 조합
2의 배수로 조합하기 쉽습니다.

<div className="code-block">{`/* 조합 예시 */
padding: var(--size-4) var(--size-8);  /* 8px 16px */
margin: var(--size-2) var(--size-4) var(--size-6);  /* 4px 8px 12px */

/* 계산 예시 */
width: calc(var(--size-8) * 3);  /* 48px */
gap: calc(var(--size-2) + var(--size-4));  /* 12px */`}</div>

## Quick Reference

| Token | px | rem | 사용 예시 |
|-------|----|----|----------|
| `--size-2` | 4px | 0.25rem | 작은 간격, 여백 |
| `--size-4` | 8px | 0.5rem | 일반 간격, 패딩 |
| `--size-8` | 16px | 1rem | 기본 간격, 마진 |
| `--size-12` | 24px | 1.5rem | 큰 간격 |
| `--size-16` | 32px | 2rem | 컨테이너 패딩 |
| `--size-24` | 48px | 3rem | 섹션 간격 |
| `--size-32` | 64px | 4rem | 큰 섹션 간격 |

| Token | px | 사용 예시 |
|-------|----|----|
| `--size-icon-sm` | 16px | 인라인 아이콘 |
| `--size-icon-md` | 20px | 버튼 아이콘 |
| `--size-icon-lg` | 24px | 헤더 아이콘 |
