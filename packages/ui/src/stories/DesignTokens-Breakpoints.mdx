# Breakpoints

반응형 디자인을 위한 브레이크포인트 토큰입니다. 일관된 미디어 쿼리를 제공합니다.

## 사용 방법

```css
@import "@lyra/design-tokens/css";

/* CSS 미디어 쿼리 */
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
@media (min-width: 1536px) { /* 2xl */ }
```

## 사용 가이드

### 기본 미디어 쿼리

<div className="code-block">{`/* Mobile First 접근 방식 (권장) */

/* 기본 스타일 - 모바일 (< 640px) */
.container {
  padding: var(--size-4);
}

/* 태블릿 이상 (≥ 768px) */
@media (min-width: 768px) {
  .container {
    padding: var(--size-8);
  }
}

/* 데스크탑 이상 (≥ 1024px) */
@media (min-width: 1024px) {
  .container {
    padding: var(--size-16);
    max-width: 1200px;
    margin: 0 auto;
  }
}`}</div>

### 범위 기반 미디어 쿼리

<div className="code-block">{`/* 특정 범위에만 적용 */

/* 태블릿만 (768px ~ 1023px) */
@media (min-width: 768px) and (max-width: 1023px) {
  .sidebar {
    width: 200px;
  }
}

/* 모바일과 태블릿만 (< 1024px) */
@media (max-width: 1023px) {
  .desktop-only {
    display: none;
  }
}`}</div>

### 반응형 그리드

<div className="code-block">{`/* 반응형 그리드 레이아웃 */
.grid {
  display: grid;
  gap: var(--size-4);

  /* 모바일: 1열 */
  grid-template-columns: 1fr;
}

/* 태블릿: 2열 */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
    gap: var(--size-6);
  }
}

/* 데스크탑: 3열 */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
    gap: var(--size-8);
  }
}

/* 큰 데스크탑: 4열 */
@media (min-width: 1280px) {
  .grid {
    grid-template-columns: repeat(4, 1fr);
  }
}`}</div>

### 타이포그래피 스케일

<div className="code-block">{`/* 반응형 폰트 크기 */
h1 {
  /* 모바일 */
  font-size: var(--font-size-xl);
  line-height: var(--font-lineHeight-tight);
}

@media (min-width: 768px) {
  h1 {
    /* 태블릿 이상 */
    font-size: var(--font-size-2xl);
  }
}

@media (min-width: 1024px) {
  h1 {
    /* 데스크탑 */
    font-size: calc(var(--font-size-2xl) * 1.25);
  }
}`}</div>

### 네비게이션 패턴

<div className="code-block">{`/* 모바일: 햄버거 메뉴 */
.nav-mobile {
  display: block;
}

.nav-desktop {
  display: none;
}

/* 데스크탑: 수평 네비게이션 */
@media (min-width: 1024px) {
  .nav-mobile {
    display: none;
  }

  .nav-desktop {
    display: flex;
    gap: var(--size-8);
  }
}`}</div>

## 디바이스 타겟팅

### 화면 크기별 일반적인 디바이스

**< 640px (모바일)**
- iPhone SE, iPhone 12/13/14 mini
- Galaxy S series (세로)
- 일반 스마트폰 (세로)

**640px ~ 767px (큰 모바일)**
- iPhone 12/13/14 Pro Max (세로)
- 큰 스마트폰 (세로)
- 폴더블 폰 (접힌 상태)

**768px ~ 1023px (태블릿)**
- iPad, iPad Air (세로)
- Android 태블릿 (세로)
- 스마트폰 (가로)

**1024px ~ 1279px (큰 태블릿 / 작은 데스크탑)**
- iPad Pro 11" (세로)
- 태블릿 (가로)
- 작은 노트북

**1280px ~ 1535px (데스크탑)**
- 일반 노트북 (1366x768, 1440x900)
- 소형 데스크탑 모니터

**≥ 1536px (큰 데스크탑)**
- Full HD 모니터 (1920x1080)
- QHD/4K 모니터
- 초광대역 모니터

## 베스트 프랙티스

### 1. Mobile First 접근

<div className="code-block">{`/* ✅ 좋음: Mobile First */
.element {
  width: 100%;
}

@media (min-width: 768px) {
  .element {
    width: 50%;
  }
}

/* ❌ 나쁨: Desktop First */
.element {
  width: 50%;
}

@media (max-width: 767px) {
  .element {
    width: 100%;
  }
}`}</div>

### 2. 의미있는 브레이크포인트

<div className="code-block">{`/* ✅ 좋음: 콘텐츠에 맞춰 조정 */
@media (min-width: 768px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* ❌ 나쁨: 모든 브레이크포인트 사용 */
@media (min-width: 640px) { /* ... */ }
@media (min-width: 768px) { /* ... */ }
@media (min-width: 1024px) { /* ... */ }
@media (min-width: 1280px) { /* ... */ }
@media (min-width: 1536px) { /* ... */ }`}</div>

### 3. Container Queries 활용

<div className="code-block">{`/* 최신 방식: Container Queries */
.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: flex;
  }
}`}</div>

### 4. rem 단위 사용

<div className="code-block">{`/* ✅ 좋음: rem 단위 (브라우저 설정 존중) */
@media (min-width: 48rem) { /* 768px */ }

/* ⚠️ 주의: px 단위 (고정 크기) */
@media (min-width: 768px) { }`}</div>

## Quick Reference

| Breakpoint | px | rem | 용도 |
|------------|----|----|------|
| `--breakpoint-sm` | 640px | 40rem | 모바일 가로 이상 |
| `--breakpoint-md` | 768px | 48rem | 태블릿 세로 이상 |
| `--breakpoint-lg` | 1024px | 64rem | 태블릿 가로, 데스크탑 |
| `--breakpoint-xl` | 1280px | 80rem | 일반 데스크탑 |
| `--breakpoint-2xl` | 1536px | 96rem | 큰 데스크탑 |

## 일반적인 레이아웃 패턴

### 1-2-3-4 그리드

```
모바일 (< 768px):     1열
태블릿 (≥ 768px):     2열
데스크탑 (≥ 1024px):  3열
큰 화면 (≥ 1280px):   4열
```

### 사이드바 레이아웃

```
모바일 (< 1024px):    스택형 (세로)
데스크탑 (≥ 1024px):  사이드바 + 메인
```

### 네비게이션

```
모바일 (< 1024px):    햄버거 메뉴
데스크탑 (≥ 1024px):  수평 네비게이션
```
