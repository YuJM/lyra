# Borders

테두리 관련 디자인 토큰입니다. 일관된 테두리 스타일을 제공합니다.

## 사용 방법

```css
@import "@lyra/design-tokens/css";

.my-card {
  border: var(--border-width-1) solid var(--color-gray-300);
  border-radius: var(--border-radius-md);
}
```

## Quick Reference

### Border Width

| Token | px | 사용 예시 |
|-------|----|----|
| `--border-width-0` | 0px | 테두리 제거 |
| `--border-width-1` | 1px | 기본 테두리, 구분선 |
| `--border-width-2` | 2px | 강조 테두리, 포커스 |
| `--border-width-4` | 4px | 액센트, 진행 표시 |

### Border Radius

| Token | Value | 사용 예시 |
|-------|-------|----------|
| `--border-radius-none` | 0px | 직각 모서리 |
| `--border-radius-sm` | 4px | 입력 필드, 작은 버튼 |
| `--border-radius-md` | 8px | 버튼, 카드 |
| `--border-radius-lg` | 12px | 큰 카드, 모달 |
| `--border-radius-full` | 9999px | 약 모양 버튼, 태그 |
| `--border-radius-circle` | 50% | 아바타, 아이콘 배지 |
